# 爱情事件

## 结婚条件

1. 和女孩/男孩发生过全部爱情事件。
2. 女孩/男孩的心是大红心。
3. 自宅扩建两次，有大床。
4. 有青色羽毛。
(女孩/男孩橙心后,信箱会收到杂货店进货通知,可购买青色羽毛)

- 拿青色羽毛求婚，一星期后在教堂举行婚礼。
（婚礼那天不能是节日，否则婚礼取消，所以要选好求婚的日子）
